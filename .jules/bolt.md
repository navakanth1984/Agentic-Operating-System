## 2024-05-24 - React Dashboard Canvas Optimization
**Learning:** High-frequency state updates in parent components (such as global mouse events, drag-and-drop operations, or typing in search palettes) will cascade re-renders down to heavy data visualization components. In this codebase, the continuous typing in the command palette and task dragging caused the `NetworkMesh3D` canvas and `SystemResourcesView` Recharts components to re-render, creating a severe bottleneck.
**Action:** Always wrap heavy visual/charting components in `React.memo`. When accepting functions as props from a parent that doesn't memoize them (like `App.tsx`), provide a custom equality function to `React.memo` that ignores the function props and only compares the data (e.g. `prevProps.metrics === nextProps.metrics`).
